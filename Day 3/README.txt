# Find the Treasure

Welcome to Find the Treasure, a classic text-based adventure game where players embark on an exciting quest to discover hidden riches on Treasure Island!

## How to Play

1. **Objective:** Your mission is to find the treasure hidden somewhere on Treasure Island.

2. **Gameplay:** You'll navigate through a series of challenges, including making decisions at crossroads, encountering obstacles like lakes and holes, and ultimately selecting the right door to unlock the treasure trove.

3. **Decisions:** At various points in the game, you'll need to make choices that will determine your fate. Choose wisely!

4. **Winning:** If you successfully navigate through all the challenges and choose the correct door, you'll find the treasure and win the game!

## Getting Started

To play Find the Treasure:

1. Clone this repository to your local machine.
2. Open the terminal and navigate to the directory containing the game files.
3. Run the game by executing the Python script: `python find_the_treasure.py`

## Game Features

- **Simple Text-Based Interface:** Enjoy an immersive gaming experience with a minimalist text-based interface.
- **Multiple Endings:** The game offers multiple possible outcomes based on the choices you make, keeping each playthrough unique and exciting.
- **Suitable for All Ages:** Whether you're a seasoned adventurer or new to text-based games, Find the Treasure offers entertainment for players of all ages.
